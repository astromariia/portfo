Kobuki Web Interface

In this project I learned how to build a web based user interface for controlling a robot. At first I learned the building blocks of HTML, then how to host my website on a flask server and finally how to get the robot to respond to input from the UI and transmit sensor data. The goal was to connect Web Programming with Embedded Software of the Kobuki robot. Overall it was a culmination of my skills as a embedded software engineer.
